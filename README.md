# AutoVision-Dashboard
About the Project
This project is a Power BI dashboard where I worked on turning raw automotive data into something meaningful and easy to understand. Instead of just looking at numbers in tables, I created interactive visuals that help quickly identify trends, patterns, and key insights.
The goal was simple — make data clear, visual, and useful for decision-making.
What I Did
Cleaned and transformed raw data using Power Query
Built interactive dashboards using Power BI
Used DAX to create calculated measures and KPIs
Designed visuals to show trends like sales performance and customer behavior.
What You Can Explore
Sales trends over time 
Performance of different vehicle categories 
Customer insights and patterns 
Key metrics displayed in a simple and interactive way.
